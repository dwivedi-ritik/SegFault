## List of commands
<ReadInfo time='1' updatetime='6 Feb 2022'/>
- Setting up samba deamon

```bash
$ sudo systemctl start smb
```

- pacman removing packages with there config files
```bash
$ sudo pacman -Rnc pkg-name
```

-  Setting default branch name in git
```bash
$ git config --global init.defaultBranch main
```
