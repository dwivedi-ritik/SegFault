## Connect to postgresql of remote machine

This is the code snippet of connecting to postgresql for remote machine. Make sure
you have installed `psycopg2'.

### Run
```shell
$ pip install psycopg2
```

### Code snippet

```python
import csv
import psycopg2

con = psycopg2.connect(
    database="your_database_name" ,
    user="user_name",
    password="password",
    host="ip_address_of_remote_machine"
)

db = con.cursor()
conn.commit()
db.close()
conn.close()
```
